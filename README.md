## Requirements
1. openssl
2. curl

## How to Install in Termux

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/adwin-15/insta-bot`

`$ cd insta-bot`

`$ chmod +x insta-bot.sh`

`$ termux-wake-lock`

`$ bash insta-bot.sh`
